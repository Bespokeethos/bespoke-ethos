## DESIGN SYSTEM (FINAL SPECIFICATIONS)

### Color Palette (Cleveland + Pride)

```css
/* Primary (Cleveland Professional) */
--color-primary-900: #1a1a2e;   /* Cleveland night sky */
--color-primary-700: #16213e;   /* Lake Erie deep water */
--color-primary-500: #0f3460;   /* Rock Hall blue */
--color-primary-300: #533483;   /* Pride purple */

/* Secondary (Cleveland Accents) */
--color-secondary-600: #e94560; /* Terminal Tower sunset */
--color-secondary-500: #f39c12; /* Autumn leaves (Cleveland fall) */
--color-secondary-400: #16a085; /* Lake Erie shoreline */
--color-secondary-300: #9b59b6; /* Pride lavender */

/* Pride Gradient (Decorative Only - Thin Borders) */
--pride-gradient: linear-gradient(135deg, 
  #e94560 0%,    /* Red */
  #f39c12 20%,   /* Orange */
  #f1c40f 40%,   /* Yellow */
  #16a085 60%,   /* Teal */
  #533483 80%,   /* Purple */
  #9b59b6 100%   /* Lavender */
);
```

**Usage Rules:**
- ✅ DO: Use pride gradient for 1-2px borders, subtle glows, section dividers
- ❌ DON'T: Use as button fills, backgrounds, or text colors
- ✅ DO: Keep Cleveland professional (navy + teal primary)
- ❌ DON'T: Overwhelming rainbow everywhere

### Typography (Modern, Performance-Optimized)

**Primary Font:** Inter Variable (Google Fonts)
- Weights: 400, 600, 700, 800
- Variable font for performance (single file, all weights)

**Fluid Typography:**
```css
--text-base: clamp(1rem, 0.95rem + 0.25vw, 1.125rem);      /* 16px → 18px */
--text-xl: clamp(1.25rem, 1.15rem + 0.5vw, 1.5rem);        /* 20px → 24px */
--text-3xl: clamp(1.875rem, 1.65rem + 1.125vw, 2.25rem);   /* 30px → 36px */
--text-5xl: clamp(3rem, 2.55rem + 2.25vw, 3.75rem);        /* 48px → 60px */
```

### Button Design (Critical Implementation)

**Pride Discount Button (25% LGBTQ+ Offer):**

**Visual Effect:**
- Base: White button, black 2px border, rounded corners (12px)
- On Hover: Rotating rainbow glow border (4s rotation)
- Trailing: Subtle glow pulse (2s)
- **NO** moving gradient inside button
- **YES** thin rotating prism border outside

**CSS Implementation:**
```css
.btn-pride-discount {
  position: relative;
  background: white;
  color: #1a1a2e;
  border: 2px solid #171717;
  border-radius: 12px;
}

.btn-pride-discount::before {
  content: '';
  position: absolute;
  inset: -4px;
  background: conic-gradient(from 0deg, 
    #e94560, #f39c12, #f1c40f, #16a085, #533483, #9b59b6, #e94560);
  border-radius: 14px;
  opacity: 0;
  filter: blur(8px);
  animation: rotate-border 4s linear infinite;
  z-index: -1;
}

.btn-pride-discount:hover::before {
  opacity: 0.7;
}

@keyframes rotate-border {
  to { transform: rotate(360deg); }
}
```

**Result:** Elegant, subtle, gorgeous (NOT tacky or overwhelming)
